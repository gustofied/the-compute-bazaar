The Compute Bazaar: Tinkering with Compute Markets

Prototype notebook · live-ish market snapshots

Market Data Layer

From scattered GPU listings to a small public price tape.

Provider APIs are sampled into bronze evidence, normalized into silver offers, and curated into gold tables for comparisons, indexes, dashboards, and eventually agents.

The first version of this market is intentionally humble. It does not yet pretend to be an exchange. It is a measuring instrument: a way to look at GPU capacity as a living price surface rather than a set of isolated product pages.

IThe Layers IIThe Index IIIThe Bazaar Table IVFutures

I The Layers

The data lake has three shelves. Bronze keeps raw provider evidence. Silver turns each provider into a shared offer schema. Gold builds the product-facing facts: listings, provider floors, index values, and the constituents supporting those values.

II The Index

A first index can be almost brutally simple: for each GPU product, find the executable floor in dollars per GPU-hour. The goal is not final methodology. The goal is a reliable path from raw observation to an auditable number.
